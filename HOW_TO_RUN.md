# 📱 Jerry Event — Client App
# HOW TO RUN ON YOUR PHONE

## STEP 1 — Install dependencies
Open VS Code terminal inside the JerryEventApp folder:
```
cd JerryEventApp
npm install
```

## STEP 2 — Start the app
```
npx expo start
```

## STEP 3 — Open on your phone
- A QR code appears in the terminal
- Open Expo Go app on your Android phone
- Tap "Scan QR Code"
- Scan the QR code → app opens instantly! 🎉

## STEP 4 — See the app
The app has 3 tabs:
- 🏠 ዋናው ገጽ — Home (services, gallery, contact)
- 📋 ቦታ ያዙ — Book an event + pay deposit
- 🔍 ያስያዙት — Track your booking by phone number
